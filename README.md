# Restaurant Rating Prediction 🍽️

## 📌 Project Overview
This project uses Machine Learning to predict restaurant ratings based on features like votes, cost, and price range.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Model Used
- Linear Regression

## 📈 Results
- MSE: 2.14
- R² Score: 0.08

## 🚀 Conclusion
The model shows basic prediction capability. Accuracy can be improved by adding more features like cuisine, location, and restaurant type.
